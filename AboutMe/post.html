My Journey Through the HTML Lab Session

Published on 1 February 2026

The first HTML lab session was an interesting and useful learning experience. In this lab, our task was to write HTML code for a simple portfolio website with two pages: a Home page and an About page. This helped me understand how a personal website is structured and created.

At the beginning, we learned the basic structure of an HTML document. Every page starts with the <html> tag, which tells the browser that the file is an HTML document. Inside it, we used the <head> tag to store page information and the <title> tag to set the name of the page shown on the browser tab. The main content of the webpage was written inside the <body> tag.

While creating the Home page, I used tags like <h1> and <h2> for headings and the <p> tag to write short paragraphs introducing myself. On the About page, I used the same tags to describe my background, interests, and goals in detail. I also learned how to use the <a> tag to create links between the Home and About pages, which made navigation easier.

One important thing I learned was how to write HTML tags as text, such as <html> and <body>, when explaining code in notes or blog posts. This helped me understand how to clearly describe HTML without confusing the browser.

Although I made a few small mistakes during the lab, fixing them helped me learn better. Seeing both my Home and About pages open correctly in the browser gave me confidence. Overall, this first HTML lab session helped me understand the basics of building a portfolio website and motivated me to learn more in future lab sessions.
